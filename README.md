### Features

- Seamless experience across all devices - desktop, tablet, and mobile
- Adaptive layouts that automatically adjust to any screen size
- Optimized touch interactions for mobile users
- Fluid typography and spacing that scales beautifully
- Toggle between light and dark themes with a smooth transition
- Eye-friendly dark mode for comfortable nighttime browsing
- Clean, modern design that looks stunning in both modes
- Detailed descriptions of all Knooz app sections
- Elegant page transitions and scroll effects
- Subtle fade-in animations for content sections
- Interactive hover effects on buttons and cards

  ### Download & Support
  - Direct APK download (Universal - works on all Android devices)
  - Detailed installation guide
  - FAQs and troubleshooting
  - Contact information for support and feedback
